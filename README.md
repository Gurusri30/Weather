Delhi Weather Data Project

This project is a simple frontend weather analysis application developed using HTML, CSS, and Vanilla JavaScript. It allows users to search Delhi weather data by selecting a specific date and view the corresponding temperature and humidity details. Additionally, users can enter a particular year to generate monthly statistics, including maximum, minimum, and median temperatures calculated from the dataset. The project uses a local JSON file (data.json) to store structured weather records containing date, month, year, temperature, and humidity information. JavaScript is used to fetch the JSON data, filter records, perform calculations, and dynamically update the HTML table with results. The application is lightweight, easy to run by simply opening the index.html file in a browser (or using Live Server if required), and serves as a practice project for understanding data handling, array operations, filtering, sorting, and DOM manipulation in JavaScript. This project can be further enhanced by integrating a real-time weather API, adding graphical charts, improving UI responsiveness, and deploying it using GitHub Pages.
